# Hidden Cost Webstory V6 - Separated Files

Versi ini adalah hasil pemisahan dari file HTML tunggal menjadi struktur proyek web statis.

## Buka proyek
Klik `index.html` untuk menjalankan langsung di browser.

## Struktur utama
- `index.html` — struktur konten HTML.
- `css/styles.css` — semua styling CSS.
- `js/script.js` — semua interaksi dan visualisasi JavaScript.
- `assets/images/` — gambar sticker/ilustrasi yang sebelumnya tertanam sebagai base64.
- `docs/ANALISIS.md` — ringkasan analisis dan perubahan.
- `docs/manifest.json` — statistik teknis hasil pemisahan.

## Hasil ekstraksi
- Gambar diekstrak: 29 file.
- CSS dipisahkan: 513 baris.
- JavaScript dipisahkan: 379 baris.
- Section HTML: 8 section.
